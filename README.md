# Dolibarr from CLI

Dolibarr from command line

Set the path where your dolibarr is installed

```
DOLPATH=/var/www/dolibarr-18.0.7/htdocs/
```

## Get users data

```
DOLPATH=/var/www/dolibarr-18.0.7/htdocs/ ./dolibarr users
```
